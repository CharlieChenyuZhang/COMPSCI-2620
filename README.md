# Client

## set up Client environment

Set up `Conda Virtual Environments`

- `conda create --name cs262-venv python=3.9`
- `conda activate cs262-venv`
- `pip install -r requirements.txt`

## how to run

`cd Client`
`streamlit run app.py`

# Server

`cd Server`
`python3 app.py`
